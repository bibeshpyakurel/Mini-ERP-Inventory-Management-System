# Security Policy

## Supported Versions

This repository is a portfolio project and does not currently publish versioned security releases.
Security fixes will be applied on the default branch.

## Reporting a Vulnerability

Do not open a public issue for suspected vulnerabilities.

Instead, report the issue privately to the repository owner with:

- a description of the issue
- steps to reproduce it
- potential impact
- any suggested remediation

The goal is to acknowledge credible reports quickly and remediate them before public disclosure.
